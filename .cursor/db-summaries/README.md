# Database snapshots

Date-stamped outputs from:

```bash
DB_HOST=mycovai.in php dev-tools/db-summary-cli.php
```

Save as `db-summary-dd-MM-yyyy.md` (e.g. `db-summary-29-05-2026.md`).

**When:** Daily and after every live database migration or bulk data change.

**Latest:** Use the most recent date in this folder for table/row counts.

See `LEARNINGS.md` and `AGENTS.md`.
